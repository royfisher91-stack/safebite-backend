## Data Collected Draft

SafeBite may collect or process the following data depending on which features a user enables:

- Account email
- Authentication/session information
- Saved products
- Favourites
- Scan/search history if enabled
- Subscription status and entitlement state
- Promo/influencer access state
- Optional allergy/profile information if enabled by the user
- Product lookup requests such as barcode or search text
- Basic support request information if a user contacts support

## Product Safety Data Limitations

SafeBite product checks depend on available product data. Ingredient, allergen, category, and safety-related fields can be incomplete, delayed, or wrong. SafeBite does not replace medical, allergy, nutrition, emergency, manufacturer, or professional advice.

## Supermarket Price/Stock Accuracy Limitations

Retailer prices, promotions, stock status, and availability can change. SafeBite displays supermarket data only where approved or controlled data is available. Users should confirm details with the retailer before purchase.

## SafeHome Add-on Limitations

SafeHome is planned as a paid add-on via `safehome_addon`. It supports household product checks but does not replace manufacturer instructions, hazard labels, emergency guidance, or professional advice.
